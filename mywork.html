Deep learning guitar player

Computer Science • Music • Coding (Python)

The aim of this work is to generate guitar music using deep learning technincs. The structure employed here may be interesting as it seems different from the examples dealing with the same objective online.
For this project web scraping technics are used to download all the available guitar parts of a selected artist from a website. Extracting the time interval between two notes and the notes we train two neural networks to predicts each one of those variables. The generated guitar tracks are then exported to a midi format.
The report is available here, the code except the scrapping part is available on this repositery..
